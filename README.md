# Онлайн-магазин с семинарами

## Описание проекта
Данный проект представляет собой онлайн-магазин, где пользователи могут:
- Просматривать товары по категориям.
- Добавлять товары в корзину.
- Создавать заказы .
- Просматривать историю заказов.
- Добавлять товары в избранное.
- Просматривать  семинары.

Проект разработан на Django  с использованием **Python 3.14**, **Tailwind CSS** для фронтенда .


## Технологии
- Python 3.14
- Django 
- Tailwind CSS (для стилизации)
- PostgreSQL (для базы данных)
- HTML, CSS, JavaScript

---

## Функциональность

### Пользовательский функционал:
- Регистрация и авторизация пользователей.
- Управление профилем (редактирование данных, смена пароля).
- Просмотр каталога товаров и детальной информации.
- Добавление товаров в корзину.
- Оформление заказов с проверкой обязательных полей.
- История заказов.
- Избранное (Favorite).
- Просмотр промо-акций и брендов.

## Установка проекта

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/your_username/online-store.git
cd online-store
```

2. **Создайте виртуальное окруже**
```
python -m venv venv
source venv/bin/activate    # для Linux/Mac
venv\Scripts\activate       # для Windows
```

3. **Установите зависимости**
```
pip install -r requirements.txt
```

4. ** Примените миграции **
```
python manage.py makemigrations
python manage.py migrate

```

5. ** Запустите сервер **
```
python manage.py runserver

```

6.  ** Откройте проект в браузере **
```
http://127.0.0.1:8000/
```

## Модели (основные)

CustomUser – расширенная модель пользователя.

Product – товар с ценой, описанием, категорией и изображением.

Category – категория товаров.

Order / OrderItem – заказ и его элементы.

CartItem – товары в корзине.

Seminars – семинары с описанием, ценой, датой и изображением.

Favorite – избранные товары пользователя.

Brands / Promo – бренды и промо-акции.


## Дополнительно

В проекте реализована адаптивная верстка на Tailwind CSS.
Используется маска для ввода телефона в форме оформления заказа.
История заказов позволяет пользователю видеть все свои покупки и сумму каждого заказа.
